Установка All Tweaker

Подготовка к установке

Перед установкой All Tweaker убедитесь, что у вас:

  • Установлен Python 3.8 или выше
  • Установлен 7-Zip для распаковки архива
  • Есть права администратора
  • Создана точка восстановления системы

Установка через установщик

Шаг 1: Скачивание

Скачайте последний релиз All Tweaker с GitHub:

  1. Перейдите на страницу релизов
  2. Скачайте файл All.Tweaker.Beta.exe

Шаг 2: Распаковка

  1. Запустите скачанный файл All.Tweaker.Beta.exe
  2. Выберите папку для распаковки
  3. Дождитесь завершения распаковки

Шаг 3: Установка

  1. Перейдите в распакованную папку "All Tweaker Beta"
  2. Запустите setup.bat от имени администратора
  3. После завершения setup.bat запустите install.bat

Установка из исходного кода

Шаг 1: Клонирование репозитория

git clone https://github.com/scode18/All-Tweaker.git
cd All-Tweaker

Шаг 2: Установка зависимостей

pip install -r requirements.txt

Шаг 3: Запуск программы

python main.py

Проверка установки

После установки:

  1. Запустите All Tweaker
  2. Проверьте, что открывается главное окно программы
  3. Убедитесь, что все вкладки загружаются корректно
  4. Проверьте работу базовых функций

Возможные проблемы

  • Если программа не запускается, проверьте права администратора
  • При ошибках с зависимостями попробуйте переустановить Python
  • Если не работают некоторые функции, проверьте права доступа к реестру

Совет

Для корректной работы всех функций рекомендуется запускать программу от имени администратора.
